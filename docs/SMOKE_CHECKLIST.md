# Smoke Checklist

Run this on a real phone or emulator before recording the demo video.

- Launch app -> Home shows the seeded open issues with color-coded status badges and Next-> strips.
- + New Issue -> composer opens ("What's going on with your appliance?") -> type a description, optionally attach a photo (Add a photo / Choose from photos) -> Start diagnosis. Confirm the description shows as your first chat bubble (with the photo inline) and the agent auto-starts, reading the photo in context. Start is disabled until the description is non-empty.
- Open an issue -> summary sheet shows the steps checklist and next step; chat continues. Reopen the app and confirm the case plus recap persist for the memory beat.
- Escalate to a pro -> Service packet screen -> guided shot list -> capture shots -> Share sheet includes the drafted message plus the mp4 when filmed.
- Trigger a dangerous request in chat -> the red safety bubble appears and the case escalates.
- Note: the live agent chat and any photo/plate read require Gemini credits. With credits depleted, demo from captured fixtures in `tests/evals/README.md` so a 429 never blocks the take.
